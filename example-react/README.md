# React Example Project

Welcome to the React Example Project! The main purpose of this project is to replicate the Notifly JS SDK and conduct testing before deployment.

## Getting Started

In the project directory, you can run the following script:

### `npm run start`

This command launches the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will automatically reload if you make any edits, and you'll see any lint errors in the console.

## Accessing Secrets

To access the environment variables (.env) for this project, please refer to [this Notion page](https://www.notion.so/greyboxhq/Dev-related-secrets-116b0f0ef3d34eb996626190677a4cd0?pvs=4). Notion access is required for retrieving the necessary information.

## Troubleshooting

If you encounter an error related to 'notifly-js-sdk-x.x.x.tgz', try running the following command at the root of the project:

```bash
npm run apply-changes-react
```

This should help resolve any issues associated with the Notifly JS SDK version. If you have further questions or face challenges, feel free to reach out for assistance.
